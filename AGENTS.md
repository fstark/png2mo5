This repo contains a set of tools to create images for the Thomson MO5 vintage computer

# Projects

- **png2mo5** — Convert any PNG image to MO5 320×200 video format (fixed 16-color palette)
- **mo5z** — Compress MO5 pixel+color banks into a single ZX0-compressed `.mo5z` file
- **mo5z2png** — Decompress a `.mo5z` file back to a 320×200 PNG
- **k7tool** — Create and inspect `.k7` cassette images for the Thomson MO5
- **mo5zviewer** — 6809 assembly viewer that decompresses and displays a `.mo5z` image from cassette
